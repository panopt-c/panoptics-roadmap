# PANOPTIC'S ROADMAP
### 18-Month AI/ML Mastery Protocol

A comprehensive, trackable roadmap for going from beginner to elite AI/ML engineer in 18 months.

## Features
- **6 phases** covering math, Python, SQL, C++, ML algorithms, deep learning, LLMs, and infrastructure
- **8 certifications** sequenced and tracked (NVIDIA, AWS, GCP, CKA)
- **Verified resources** with links to free courses, docs, and tutorials on every track
- **Compound learning methodology** built in (Active Recall, Spaced Repetition, Feynman Technique, Interleaving)
- **Progress tracking** via localStorage — persists across sessions, unique per user
- **Dark/Light mode**
- **Editable name** — anyone can personalize it
- **Zero dependencies** — pure HTML/CSS/JS, works offline

## How to Use
1. Visit the live site above, or download `index.html` and open in any browser
2. Click your name to personalize it
3. Click phases to expand, check off items as you complete them
4. Your progress saves automatically in your browser

## Setup Your Own GitHub Pages
1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://YOUR-USERNAME.github.io/panoptics-roadmap`
